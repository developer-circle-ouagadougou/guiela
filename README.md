# GUIELA
La mise en place du backend de l'API.
Pour rappel, c'est un API de raprochement bancaire.


## Requirements

* Node JS >= 12
* NPM >= 6

## Testing
1. Cloner le projet et exécuter la commande : 
  ```shell
  $ npm install
  ```
2. Exécuter le fichier index.js
  ```shell
  $ node index.js
  ```
3. Ajouter votre contribution au code
